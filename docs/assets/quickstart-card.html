Quick Start

From local Codex task to iPhone push.

Three commands to set up Bark, start the monitor, and open the dashboard.
1. Configure Bark Generate local config with your Bark device URL.
2. Start the monitor Watch local Codex `task_complete` events in the background.
3. Open the dashboard See status, recent logs, today count, and send a test notification. 
powershell -ExecutionPolicy Bypass -File .\Setup-CodexMonitor.ps1 -BarkUrl "https://api.day.app/your-device-key/"

powershell -ExecutionPolicy Bypass -File .\outputs\codex-task-monitor\Start-CodexTaskMonitor.ps1

powershell -ExecutionPolicy Bypass -File .\outputs\codex-task-monitor\Start-CodexTaskMonitorDashboard.ps1
