# v0.1.1

Windows desktop GUI release for Codex iPhone Notifier.

## Highlights

- Adds a practical WPF desktop GUI for daily Windows use
- Includes tray behavior, startup toggle, status summaries, diagnostics, and recent logs
- Ships a packaged `ps2exe` desktop build for simpler double-click startup
- Keeps the local Bark-based notification flow and Codex session monitor path intact

## Included in this release

- Bark sender with local config support
- Codex task monitor with persisted state tracking
- Local dashboard and snapshot export flow
- WPF desktop GUI
- Windows startup installer and uninstaller
- Health-check and setup scripts
- GitHub-ready docs, release notes, and MIT license

## Recommended package

For normal Windows use, start with:

- `CodexMonitorGui-ps2exe-win64.zip`

After extracting, run:

- `CodexMonitorGui.exe`

## Notes

- This release keeps the current `outputs/` layout to avoid breaking a known-good working setup
- Bark remains the default iPhone path because it proved more reliable than `ntfy` in this workflow
- Release assets include template config only, with no local Bark URL, PID files, logs, or archived runtime state
- Future releases may further clean up packaging and directory layout without changing the core monitor behavior
