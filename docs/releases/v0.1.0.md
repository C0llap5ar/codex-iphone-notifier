# v0.1.0

First public release of Codex iPhone Notifier.

## Highlights

- Reliable iPhone push notifications for Codex Desktop task completion using Bark
- Local Windows monitor that watches Codex session logs instead of depending on flaky desktop hook paths
- Lightweight dashboard with monitor status, recent logs, "completed today" count, and a one-click test notification button
- Setup, health-check, and Windows startup scripts for easier local installation

## Included in this release

- Bark sender with config template support
- Codex task monitor with persisted state tracking
- Local dashboard and snapshot export flow
- GitHub-ready docs, issue templates, PR template, changelog, and MIT license

## Notes

- This release intentionally keeps the current `outputs/` layout to avoid breaking a known-good working setup
- Local secrets and runtime files are ignored by default
- A future release may clean up the directory layout further without changing the core behavior

## Recommended first steps

1. Run `Setup-CodexMonitor.ps1` with your Bark URL.
2. Run `Test-CodexMonitor.ps1`.
3. Start the monitor and open the dashboard.
