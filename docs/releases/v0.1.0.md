# v0.1.0

First public release of Codex iPhone Notifier.

## Highlights

- Reliable iPhone push notifications for Codex Desktop task completion using Bark
- Windows-first local monitor that watches Codex session logs instead of relying on fragile hook paths
- Desktop GUI for daily use, including start/stop controls, settings, diagnostics, and recent logs
- Lightweight local dashboard for quick visibility into monitor state
- Packaged `ps2exe` desktop build for easier double-click use on Windows

## Included in this release

- Bark sender with local config support
- Codex task monitor with persisted state tracking
- Local dashboard and snapshot export flow
- WPF desktop GUI
- Windows startup installer and uninstaller
- Health-check and setup scripts
- GitHub-ready docs, release notes, and MIT license

## Recommended package

For normal Windows use, start with:

- `CodexMonitorGui-ps2exe-win64.zip`

After extracting, run:

- `CodexMonitorGui.exe`

## Notes

- This release keeps the current `outputs/` layout to avoid breaking a known-good working setup
- Bark is the default iPhone path because it was more reliable than `ntfy` on this workflow
- Release assets should include template config only, with no local Bark URL, PID files, logs, or archived runtime state
- Local secrets and runtime state are ignored by default in the repository
- Future releases may clean up the directory layout further without changing the core notification flow
